# Briefly AI 🧠📰

> News, Briefed & Visualized — A smart multimodal news assistant.

## Overview
Briefly AI is a Flutter-based mobile application that solves information 
overload by summarizing, translating, and visualizing news using AI.

## Features
- 📝 Multimodal input (Text, Voice, OCR Camera)
- 🤖 AI-powered summarization & translation
- 📊 InsightLens analytics dashboard
- 🌐 Personalized news feed
- 📁 Offline history & bookmarks

## Tech Stack
| Layer | Technology |
|---|---|
| Frontend | Flutter (Material 3) |
| State Management | Riverpod |
| Backend | Python FastAPI |
| AI Models | Hugging Face (BART, MarianMT, VADER) |
| Local Storage | Hive |
| Charts | fl_chart |

## Screens
- Splash / Onboarding
- Home Dashboard
- Chat Interface
- Camera / OCR Scanner
- InsightLens Analytics
- History & Bookmarks
- Article Detail
- Settings & Profile

## Project Structure
```
lib/
  screens/      # All app screens
  models/       # Data models
  services/     # API, OCR, Speech services
  widgets/      # Reusable UI components
  providers/    # State management
  utils/        # Theme, constants, helpers
```

## Setup
```bash
flutter pub get
flutter run
```
